Dataphi HR Analytics Dashboard is a data-driven dashboard project designed to help analyze and visualize HR metrics such as **employee headcount**, **attrition**, **department-wise distribution**, and other key workforce indicators.  
It enables quick, clear, and actionable insights to support HR and business decision-making.

---

## Key Features

- **Headcount Overview**: View total employees and workforce distribution.
- **Attrition Analysis**: Track attrition trends and identify high-risk segments.
- **Department / Role Breakdown**: Compare teams and roles using visual summaries.
- **Demographic Insights**: Explore employee distribution by key categories *(if included in your dashboard)*.
- **Interactive Filters**: Slice the data by department, role, gender, age group, etc. *(based on your report)*.
- **Clean Visual Storytelling**: Easy-to-understand charts for quick HR insights.

---

## Technology Stack


- **Dashboard Tool:** Power BI Desktop *(or Excel / Tableau)*
- **Data Format:** CSV / Excel *(as used in your project)*
- **Other:** Data Cleaning, Data Modeling, DAX Measures *(if Power BI)*

## Why This Dashboard?

- **Clarity:** Converts raw HR data into meaningful visuals.
- **Efficiency:** Saves time by centralizing key HR KPIs in one place.
- **Decision Support:** Helps HR teams identify trends, risks, and improvement areas.
- **Impact:** Enables better hiring, retention, and workforce planning decisions.

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Kuldipgodase07/Dataphi-HR-Analytics-Dashboard.git
cd Dataphi-HR-Analytics-Dashboard
```

### 2. Open the Dashboard

#### If this is a Power BI project:
1. Install **Power BI Desktop**
2. Open the `.pbix` file from the repository
3. If required, update the dataset file path
4. Click **Refresh** to load the data
